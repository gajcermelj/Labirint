# 🏴‍☠️ The Lost Pirate

Igra, zgrajena z HTML5, CSS in JavaScriptom. Igraš kapitana Barbosso, ki je po uničujočem viharju obtičal na otoku okostnjakov. Najdi pot skozi labirint, zberi vseh 30 zlatih kovancev in poišči zaklad — preden ti zmanjka časa!

---

## 🎮 Igranje

- **Cilj:** Zberi 30 zlatih kovancev po labirintu in doseži skrinjico z zakladom
- **Časovna omejitev:** 150 sekund — izgubiš!
- **Upravljanje:** Puščične tipke za premikanje
- Skrinjica z zakladom se animira, ko zbirate kovance

---

## 🛠️ Tehnologije

- **HTML5 Canvas** — risanje labirinta in animacija lika
- **JavaScript** — zanka igre, zaznavanje trkov, pobiranje kovancev
- **CSS** — oblikovanje in ozadja

---

## 🚀 Kako zaženem igro

1. Kloniraj ali prenesi repozitorij
2. Odpri `zacetnastran.html` v brskalniku
3. Klikni **Start** za začetek ali **Instruction** za zgodbo
